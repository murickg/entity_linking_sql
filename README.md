## Что это значит

**Наш подход выигрывает по F1**: мы предсказываем **в 5 раз меньше таблиц** и **в 10 раз меньше колонок**, при этом качество F1 в **2-3 раза выше**. То есть наша схема — намного "чище".

**Официальный AutoLink выигрывает по SRR**: они почти всегда (96%) находят все нужные элементы, но завалены мусором.

## Какой подход "правильный" зависит от задачи

| Применение | Что важнее | Кто лучше |
|---|---|---|
| **Чистое schema linking** (твоя задача) | F1 | Наш |
| **Schema linking → SQL gen** (их задача) | SRR + recall | Официальный |
| **Малый context window LLM** | Compact precision | Наш |
| **Большой context, downstream фильтрует** | Высокий recall | Официальный |
